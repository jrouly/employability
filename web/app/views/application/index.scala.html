A note about course descriptions

The number of raw course and job descriptions roughly correspond. However, the data quality of the course descriptions was clearly inferior compared to the job descriptions. Raw, unmodified course descriptions were downloaded from primary sources (university catalog websites). Unfortunately, the average course description is a very brief description of the mechanics of a course. The course description that details expectations, concepts, and outcomes of a course is a rare one. A more appropriate data source, though significantly less available, would be course syllabi.
